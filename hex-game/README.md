# Hex Game

A two-player, same-screen implementation of the classic board game Hex using Phaser 3 + Vite.

## Local development

```bash
cd hex-game
npm install
npm run dev
```

## Production build

```bash
cd hex-game
npm install
npm run build
```

The production build outputs to `hex-game/dist/`.
